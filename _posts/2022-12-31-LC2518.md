## Leetcode 2518 Hard - Number of Great Partitions

### Question

Given an array `nums` and int `k`, partition the array into two ordered groups s.t. each element is in exactly **one** group. A partition is called great if the sum of the elements is $\geq$ `k`. Return the number of *distinct* great partitions modulo $10^9 + 7$. 

Two partitions are considered distinct if some element is different in the two partitions.
